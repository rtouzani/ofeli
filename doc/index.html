ofeli is an open-source software package containing:

  • A library of c++ classes (ofeli: Object Finite Element Library) to solve partial differential equations by finite element methods. It also contains numerous classes and functions to solve various numerical analysis problems.
  • A line command application (rita) to solve various numerical simulation problems.
    It handles:
    • Interpolation and function approximation (Lagrange, Spline, Least squares, Nurbs, Bézier)
    • Algebraic Equations
    • Ordinary and Partial Differential Equations
    • Optimization problems
    • Eigenvalue problems

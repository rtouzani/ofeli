Using external libraries

The OFELI library can be used in conjunction with the following external libraries:

PETSc: Portable, Extensible Toolkit for Scientific Computation
    PETSc is a library of functions for the parallel solution of scientific applications.
Eigen: A template C++ library for linear algebra.

Using PETSc

Once you have installed the PETSc library and a chosen MPI library for parallel computation, you can switch to its use in conjunction with OFELI by the cmake script.

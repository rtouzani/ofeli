Detailed Description

A simple class to parse real valued functions.

Functions must have 1, 2, 3 or at most 4 variables.

Warning
Data in the file must be listed in the following order: 
   for x=x_0,...,x_I
       for y=y_0,...,y_J
           for z=z_0,...,z_K
               read v(x,y,z)
